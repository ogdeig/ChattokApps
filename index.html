What to write

If you’re vague, you’ll get a bland game. These hints help.

Choose the “type” without saying template Say what it does: trivia, wheel spin, battle, racing, voting, scoreboard overlay, etc.
One “main loop” Example: “join → play rounds → show winner → reset” or “always running scoreboard overlay”.
Big UI for phones Ask for bold fonts, high contrast, and clear spacing. (Your viewers are on mobile.)
Monetization-safe wording Avoid “pay to win”. Instead: “optional boosts”, “bonus effects”, “fun power-ups”.
